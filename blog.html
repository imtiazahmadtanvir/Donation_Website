What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of the document as a tree of objects, allowing you to manipulate its content and visual presentation. Each object represents a part of the document, such as an element or attribute.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like `document.getElementById()` for selecting by ID, `document.querySelector()` for selecting by CSS selectors (classes, tags, etc.), or `document.getElementsByClassName()` for selecting by class name. These methods allow dynamic manipulation of the selected elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events on its child elements. Instead of adding multiple listeners, the parent handles events as they bubble up. This improves efficiency and simplifies dynamic element handling.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, use methods like `setAttribute()` or `element.attribute = value`. For styles, use `element.style.property = value`. For example, to change an image's `src`, use `img.setAttribute('src', 'image.jpg')`, and for styles: `element.style.color = 'red';`.
